# Sales Performance Analysis Dashboard

## Project Overview
This project analyzes Superstore sales data using SQL, Python, and Power BI.

## Tools Used
- SQL (MySQL)
- Python (Pandas, Matplotlib)
- Power BI

## Dashboard Features

### Page 1: Sales Performance Dashboard
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Sales Trend Analysis
- Sales by Category
- Sales by Segment
- Profit by Region

### Page 2: Customer & Product Analysis
- Average Order Value
- Profit Margin
- Total Customers
- Sales Distribution by State
- Top 10 Products
- Top 10 Customers
- Discount vs Profit Analysis

## Key Insights
- Technology is the highest revenue-generating category.
- West region generates the highest profit.
- Consumer segment contributes the largest share of sales.
- High discounts negatively impact profitability.
- A small group of customers contributes significantly to revenue.
